# Rapporti PNG → PG (Tracker)

> [!note] A cosa serve
> File **canonico** per il punteggio di rapporto dei PNG importanti verso il gruppo. Aggiornare **qui** ogni volta che un'azione dei PG sposta una relazione, e tenere allineato lo snapshot in [[campaign_state|context/campaign_state.md]].

## Scala

| Punteggio | Significato |
| --- | --- |
| **+3** | Alleato leale (si espone per i PG) |
| **+2** | Favorevole (aiuta volentieri) |
| **+1** | Ben disposto (un favore, uno sconto, fiducia iniziale) |
| **0** | Neutro / non ancora definito |
| **-1** | Diffidente / deluso |
| **-2** | Ostile (rema contro, nuoce se può) |
| **-3** | Nemico dichiarato |

## Come Aggiornare

- A fine sessione (o quando cambia qualcosa): **modifica il punteggio qui** e annota in una riga il **perché**.
- Aggiorna lo **snapshot** in [[campaign_state|context/campaign_state.md]] e aggiungi una riga a [[log|context/log.md]] e [[migration_state|context/migration_state.md]].
- Aggiungi nuovi PNG quando entrano stabilmente in rapporto col gruppo (es. [[Elira]], [[Dario]], [[Tommaso]] quando partono i loro archi).
